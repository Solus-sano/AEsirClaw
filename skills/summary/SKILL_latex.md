# Skill Description
当需要总结历史消息时使用此技能。自动提取历史消息并生成摘要。

## Usage

1. 调用获取历史信息的工具，获取历史消息列表
2. 根据获取到的信息进行总结，将内容总结为一个或多个话题，以及各个群友的称号（6个字以内），生成总结内容时，你需要严格遵守以下准则：
   - 每个话题的内容应该比较清晰，不能太笼统
   - 明确主体与动作：在每条总结中点名具体群友（或群体）在何时做了什么，以及对话题产生的具体影响；避免与群友行为无关的泛化表述。
   - 交代脉络与结论：尽量给出起因→经过→结果，并在首句给出清晰结论。
   - 标注信息来源：引用观点或数据时，尽量指明发言人、时间或关键信息片段，提升可追溯性。
3. 将总结结果写到tex文件并编译成 pdf，并将文件发送出来
4. 发送文本“总结已经发送，请查看pdf文件喵～”

【编译流程】
1. 文档类使用 ctexart（ctex宏包套件的中文文章类）
2. 编译引擎必须用 XeLaTeX 或 LuaLaTeX，pdflatex 不支持系统字体
3. 执行命令：xelatex -interaction=nonstopmode filename.tex
4. ctex会自动检测系统并选择合适的中文字体（Linux默认用Fandol字体集）


## latex编译流程
【相关依赖】（若没有安装，请先自行安装）
✓ XeTeX 3.141592653 (TeX Live 2025/dev/Debian)
✓ ctex 宏包 v2.5.10（中文支持核心）
✓ xeCJK 宏包（中日韩文字处理）
✓ fontspec 宏包（系统字体调用）
✓ Fandol 字体集（开源中文字体）

【关键配置】
- 文档类声明：\documentclass[UTF8]{ctexart}
- 无需手动指定字体，ctex自动配置
